
## Project Setup

##Node Version
```node v 21```

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Database setup

```sh
php artisan migrate
```
### Seeders
```sh
php artisan db:seed --class=RolesPermissionsSeeder
```
